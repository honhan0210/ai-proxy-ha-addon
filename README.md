# AI Proxy Home Assistant Add-on Repository

This repository contains a custom Home Assistant add-on named `AI Proxy`.

## Contents

- `ai_proxy/`: Home Assistant add-on source
- `repository.yaml`: Repository metadata for Home Assistant import

## Import

1. Push this folder to `https://github.com/honhan0210/ai-proxy-ha-addon`.
2. In Home Assistant, open the Add-on Store.
3. Add this repository URL:

```text
https://github.com/honhan0210/ai-proxy-ha-addon
```
